🔒 Конфиденциальность
Прежде чем начать
Приложение использует данные для персонализации прогноза. Вот что именно и зачем:
📍Город и геолокация — для получения погоды. Хранится только на устройстве, не на сервере.
👗Стиль одежды, активность, транспорт — для персонализации советов. Не передаются третьим лицам.
💊Данные о здоровье — только локально для погодных предупреждений. На сервер не отправляются никогда.
🌤Запросы к OpenWeatherMap — содержат только название города, без личных данных.
📢Реклама — показывается бесплатным пользователям без персонального таргетинга.
Продолжая использование, вы соглашаетесь с Политикой конфиденциальности
